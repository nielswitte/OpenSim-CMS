OpenSim-CMS

For a full documentation of the used API see the API section. This CMS allows you to use the functions of the API to set up a meeting in the virtual environment of OpenSim. This API and CMS is part of my Master Thesis with the subject of "Using a Content Management System to Increase the Usability of a 3D Virtual Environment".

The system is based on a client server setting. The server provides a RESTful JSON API with a back-end build in PHP and MySQL. The client front-end is build in AngularJS with the Restangular service and Boostrap combined with support from Angular-Strap, jQuery, LessCSS and others. See the CMS documentation for more information.
