OpenSim

Wednesday, April 16, 2014 11:10 AM

OpenSim needs to be configured with the following settings:

For loading dynamic textures:

[XEngine]
    AllowOSFunctions = true

Enable JSON support:

[XEngine]
    AllowMODFunctions = true
[JsonStore]
    Enabled = true

For RemoteAdmin functions:

[RemoteAdmin]
    enabled = true
    port = 9000
    access_password = "<ACCESS PASSWORD HERE>"
    access_ip_addresses = 127.0.0.1
    enabled_methods = all

The cache functions used by presenterScreen.lsl require the use of the FlotsamCache.ini. For more information about the FlotsamCache see:
http://opensimulator.org/wiki/AssetCache.

In addition it is recommended to use MySQL as a database server for OpenSim. See http://opensimulator.org/wiki/Database_Settings#MySQL_Walkthrough for instructions on how to set things up.
